Tutto su di me

Benvenuto, mi chiamo Federico Calò e sono un Software e Web Developer

La mia passione per l'informatica nasce tra i banchi delle scuole superiori e, dopo essermi diplomato, ho iniziato gli studi universitari presso la facoltà di Informatica dell'università Aldo Moro di Bari. Fin da subito ho applicato le mie conoscenze per creare siti web e applicazioni. Per migliorare le mie competenze, creo sistemi gestionali e siti web con diversi framework quali Spring Boot, React, Angular. Frequento corsi specifici per tenermi sempre aggiornato sulle nuove tecnologie. I linguaggi che più utilizzo sono:

  • Java per le applicazioni desktop e i gestionali;
  • Python per programmi di Data Analysis e intelligienza Artificiale;
  • C, C++, C# per applicazioni e giochi;
  • HTML, CSS e JavaScript per la creazione di siti web;
  • MySQL e MongoDB per la gestione di basi dati relazionali e non relazionali.

Come framework per la creazione di applicazioni web e desktop, invece, sto migliorando le mie capacità con:

  • Spring Boot per la creazione del Back End delle applicazioni;
  • React e Angular per la creazione del Front End delle applicazioni;
  • Lo stack MERN per creare applicazioni web
  • JavaFX per creare la grafica di applicazioni desktop.

Inoltre sto frequentando corsi di formazione per sviluppare la programmazione in cloud. Infine, come potrete vedere più avanti, lavoro anche per una società di sviluppo software che mi permette di testare sul campo le mie conoscenze e competenze. Per qualsiasi esigenza non esitate a contattarmi!!!
